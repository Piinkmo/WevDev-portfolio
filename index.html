    • Back_5.3 Axios(Jul 20, 2024)

    Axios is a promise-based HTTP client for JavaScript, used both in the browser and Node.js.

  • Promise-based: Makes handling asynchronous HTTP requests simpler.
  • Interceptors: Allows you to intercept and modify requests or responses.
  • Automatic JSON Data Transformation: Automatically transforms JSON data.
  • Request Cancellation: Supports cancelling requests.
  • Browser Compatibility: Works with all modern browsers.
  • XSRF Protection: Provides protection against cross-site request forgery.

    Capstone Project 2 JokeAPI 2(Jul 28, 2024)

    Hosted on localhost:3000

    Overview

    The Personalized Joke Generator is a web app built with Express.js and Node.js. It uses JokeAPI to fetch jokes and personalize them with the user's name.

    Features

  • User Input: Users enter their name.
  • Personalized Jokes: Fetches a random joke and replaces "Chuck Norris" with the user's name.
  • Error Handling: Displays user-friendly error messages.

    • Technical Details

  • Backend: Node.js with Express.js.
  • HTTP Client: Axios.
  • Templating Engine: EJS.

app.delete("/jokes/:id", (req, res) => {
  const id = parseInt(req.params.id);
  const searchIndex = jokes.findIndex((joke) => joke.id == id);
  if (searchIndex > -1) {
    jokes.splice(searchIndex, 1);
    res.sendStatus(200);
  } else {
    res
    .status(404)
    .json({error: `Joke with id: ${id} not found. No jokes were deleted.`});
  }
});
